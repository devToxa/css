HTML5 기본 문서 구조: <!DOCTYPE html>과 웹 브라우저 탭 제목("id 선택자")을 포함하여 표준적인 웹 페이지의 메타 정보와 기본 틀을 정의했습니다.

태그 및 ID 선택자 지정: <style> 태그 안에 모든 <p> 태그를 빨간색(red)으로 만드는 태그 선택자와, ctxt라는 고유 ID를 가진 요소를 파란색(blue)으로 만드는 ID 선택자(#ctxt)를 선언했습니다.

선택자 우선순위 적용 결과: 본문에서 첫 번째 단락은 태그 스타일에 따라 빨간색, 두 번째 단락은 태그 스타일보다 우선순위가 높은 ID 스타일이 적용되어 파란색으로 출력됩니다.

스타일 미적용 단락 출력: 세 번째 단락(<p id="txt">)은 #ctxt 스타일에 해당하지 않으므로 기본 태그 선택자의 영향을 받아 빨간색으로 화면에 출력됩니다.
