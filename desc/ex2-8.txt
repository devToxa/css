HTML5 기본 문서 구조: <!DOCTYPE html> 설정과 브라우저 탭 제목("전체 선택자")을 포함하여 표준적인 웹 페이지의 메타 정보와 기본 틀을 정의했습니다.

전체 선택자(Universal Selector) 사용: <style> 태그 안에 별표(*) 모양의 전체 선택자를 사용하여 문서 내의 모든 HTML 요소에 스타일을 지정했습니다.

일괄 스타일 적용: 전체 선택자에 color: blue;를 선언함으로써, 웹 페이지 안에 포함된 모든 글자 색상을 일괄적으로 파란색으로 설정했습니다.

본문 내용 파란색 출력: 실제 본문(<body>)에 있는 제목(<h2>), 리스트 목록(<ul>, <li>), 단락(<p>) 등의 태그 종류나 클래스명에 상관없이 모든 텍스트가 파란색으로 화면에 출력됩니다.
