동적 클래스 설계: CSS에 하이라이트 효과를 주는 .active-blue와 .active-red 클래스를 미리 정의해 두었습니다.

이벤트 바인딩: 각 버튼의 onclick 속성에 applySelector() 함수를 연결하여 클릭 시 특정 선택자와 색상 값을 전달합니다.

선택자 전달: 버튼을 누르면 h3[class]나 p[class*="icox"] 같은 CSS 선택자 문자열이 자바스크립트 함수로 전달됩니다.

기존 상태 초기화: 새로운 버튼을 누를 때마다 resetAll() 함수가 먼저 실행되어 이전에 적용되었던 하이라이트 효과를 모두 지웁니다.

DOM 탐색 (querySelectorAll): 브라우저의 document.querySelectorAll() 메서드를 사용해 전달받은 CSS 선택자와 일치하는 HTML 요소를 똑같이 찾아냅니다.

범위 제한: 전체 페이지가 아닌 실습 구역인 #sandbox 내부의 요소들만 탐색하도록 범위를 제한하여 안전성을 높였습니다.

반복문 적용 (forEach): 선택자와 일치하여 찾아낸 일치 항목(배열 구조)을 루프를 돌며 하나씩 접근합니다.

조건별 클래스 추가: 함수가 전달받은 인자(color)가 'blue'면 파란색 클래스를, 'red'면 빨간색 클래스를 요소의 classList에 추가합니다.

실시간 렌더링: 자바스크립트에 의해 클래스가 추가되는 순간, 브라우저가 CSS 스타일을 실시간으로 다시 계산해 화면에 반영합니다.

독립된 초기화: '초기화' 버튼은 스타일 추가 없이 resetAll()만 호출하여 언제든지 처음 깨끗한 상태의 본문으로 되돌립니다.
