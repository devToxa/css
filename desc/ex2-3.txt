HTML5 기본 문서 구조: <!DOCTYPE html>과 <html lang="en">을 사용하여 웹 표준에 맞춰 작성된 영어 기준의 웹 페이지 문서입니다.

메타 정보 및 제목 설정: 브라우저 화면 크기에 맞춘 화면 반응형 설정(viewport), 글자 깨짐 방지(UTF-8), 그리고 웹 브라우저 탭에 표시될 제목("스타일 시트 포함")을 지정했습니다.

외부 CSS 파일 불러오기: <style> 태그 내의 @import 명령어를 사용하여 상위 디렉토리의 css 폴더에 있는 ex2-3.css 외부 스타일시트 파일을 가져와 연결했습니다.

본문 내용 출력: 웹 페이지의 실제 본문(<body>)에 <p> 태그를 사용하여 "우리를 기쁘게 하는 것들."이라는 문장을 단락으로 화면에 출력합니다.
